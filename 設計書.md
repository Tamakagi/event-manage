テーブル定義
jobs
| カラム | 必須 | 用途 |
| ---------- | -- | -------------------------------------------- |
| id | ○ | ジョブID |
| type | ○ | `MESSAGE`、将来は `REMINDER` など |
| channel_id | × | 投稿先チャンネル |
| content | × | 投稿内容 |
| execute_at | ○ | 実行予定日時 |
| status | ○ | `WAITING` / `RUNNING` / `SUCCESS` / `FAILED` |
| message_id | × | 投稿後にDiscordから返るID |
| created_at | ○ | 作成日時 |
| updated_at | ○ | 更新日時 |
